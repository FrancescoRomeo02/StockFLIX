Icone
Tutte le icone del sito sono state reperite da FontAwesom un sito free use che, tramite la sua libreria Javascrip rende possibile accedere a moltissime Icone come se fossero semplice testo.
Diagrammi
Tutti i diagrammi della pagina sono stati realizzati con la libreria amcharts, una potente libreria che con pochi e semplici comandi permette di realizzare i più svariati diagrammi. In particolare per questa pagina ho utilizzato i diagrammi appositamente creati per il mondo della finanza passando i dati tramite i file csv.
Disegni
I disegni presenti sulla pagina sono dei file SVG che ho scaricato dal sito web UNdraw, un sito web che mette a disposizione moltissimi file SVG per i più svariati usi, grazie al sito è anche possibile una minima personalizzazione delle immagini.
Python
Le librerie utilizzate per il progetto sono moltissime, in particolare posso citare ftplib per la comunicazione ftp con il server, alpha_vantage per ottenere i dati aggiornati delle azioni, numpy per la manipolazione dei dati e pandas con lo stesso scopo.
